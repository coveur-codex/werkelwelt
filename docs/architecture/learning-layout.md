# Didaktisches Layout der Plus-Werkstatt

Die Plus-Werkstatt ist ein Lernarbeitsplatz, kein Dashboard. Für die aktive Addition gelten diese Layoutregeln:

1. Aufgabe, Material und aktueller Schritt müssen gleichzeitig sichtbar sein.
2. Nur der aktuelle Schritt ist prominent.
3. Feedback ist kurz.
4. Keine Abschlussfragen während aktiver Rechnung.
5. Eingabehilfen dürfen den Rechenkern nicht verdrängen.
6. Stellenwerte dürfen platzsparend abgekürzt werden, müssen aber erklärbar sein.

Stellenwerte werden zentral mit `E`, `Z`, `H`, `T`, `ZT`, `HT` und `M` abgekürzt. Die ausgeschriebene Bedeutung bleibt über Tooltip, Tastaturfokus und Accessibility-Labels erreichbar.
